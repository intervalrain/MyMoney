# MyMoney

簡易的記帳 App

## Introduction
+ 具備收入、支出、轉帳的記帳功能
+ 具備計算機作為輸入介面
+ 具備 summary page 顯示本年、本月、本週、本日收入支出
+ 具備不同類型帳戶(Cash, Bank) 顯示總額
+ 每個帳戶可以顯示細項
